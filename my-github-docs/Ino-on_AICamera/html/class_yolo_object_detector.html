Run YOLOv4 inference with YoloObjectDetector::frame image. Result image with detected boxes and inference time is saved into YoloObjectDetector::frame. YoloObjectDetector::setModel() should be called first.

Parameters
targetClass ID of target object in your .names file.
confThresholdConfidence threshold of Yolo. It should be 0 to 1.
nmsThresholdNMS threshold of Yolo. It should be 0 to 1.
resizeThe size of the image input to the top network. It should be multiples of 32.
